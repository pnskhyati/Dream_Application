# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/117I_bEAE8m_iveO6RBZSeRNDEUfvHsxp

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`


The solution is a Personalized Career and Skills Advisor powered by Google Cloud’s Generative AI, designed to transform how Indian students navigate their academic and professional journeys. Unlike generic career guidance platforms, this advisor delivers deeply tailored insights by analyzing each student’s unique profile—including academic performance, interests, strengths, and future aspirations.  Beyond suggesting suitable career paths, the platform provides a clear, step-by-step roadmap of actionable skills, certifications, and projects required to excel in today’s rapidly evolving job market. By leveraging Generative AI, it continuously adapts to shifting industry demands and emerging career opportunities, ensuring that students are always equipped with future-ready skills.  This approach empowers students to make informed, confident, and strategic career choices, bridging the gap between education and employability while addressing the specific challenges of India’s dynamic workforce landscape.
